
# websafe-uid [![Build Status](https://travis-ci.org/timshadel/websafe-uid.png?branch=master)](https://travis-ci.org/timshadel/websafe-uid)

  Generate websafe random strings.

## Example

```js
var websafe = require('websafe-uid');

websafe.uid(10);
// => "FDaS435D2z"
```
